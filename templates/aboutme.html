About Trade Evaluation

The main purpose of this project is to provide investors a set of tools for analyses and visualization of investment performance.

Managing Hypothetical Portfolio

Concept: For a new account open, user is provided with hypothetical portolios of cash for each of the markets that the application supports, namely Germany, the U.S., and Vietnam.
Users can implement their own trading strategies, making decisions on buying or selling which stocks. The application will help them evaluate and visualize the performance of their portfolio.
